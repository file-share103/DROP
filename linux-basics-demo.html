Linux Basics: Essential Commands

Educational Note: This guide provides essential Linux commands for beginners, especially useful for cybersecurity and ethical hacking tasks in Kali Linux.

1 Introduction to Linux

Linux is a powerful, open-source operating system widely used in cybersecurity, server environments, and ethical hacking. Kali Linux, a specialized distribution, comes pre-loaded with hundreds of security tools for penetration testing and digital forensics.

Understanding basic Linux commands is essential for:

  • Navigating the file system efficiently
  • Managing files and directories
  • Monitoring system resources
  • Networking and security operations
  • Running security tools and scripts
Kali Linux Terminal
kali@kali:~$ whoami
kali
kali@kali:~$ uname -a
Linux kali 5.10.0-kali9-amd64 #1 SMP Debian 5.10.46-4kali1 (2021-08-09) x86_64 GNU/Linux

2 File & Directory Navigation

Navigating the Linux file system is one of the most fundamental skills. Here are the essential commands:

Command Description
pwd Show current directory
ls List directory contents
cd <dir> Change directory
cd .. Go up one directory
mkdir <dir> Create new directory
rmdir <dir> Remove empty directory
rm -r <dir/file> Delete file/folder
touch <file> Create empty file
File Navigation Example
kali@kali:~$ pwd
/home/kali
kali@kali:~$ mkdir test_dir kali@kali:~$ cd test_dir kali@kali:~/test_dir$ touch sample.txt kali@kali:~/test_dir$ ls
sample.txt
kali@kali:~/test_dir$ cd .. kali@kali:~$ rm -r test_dir

3 File Operations

Working with files is a common task in Linux. Here are the key commands:

Command Description
cp file1 file2 Copy file
cp -r dir1 dir2 Copy directory recursively
mv old new Rename or move file/folder
cat file.txt View file content
less file.txt View with scroll (press q to quit)
File Operations Example
kali@kali:~$ echo "This is a test file" > test.txt kali@kali:~$ cat test.txt
This is a test file
kali@kali:~$ cp test.txt backup.txt kali@kali:~$ ls
backup.txt test.txt
kali@kali:~$ mv test.txt renamed.txt kali@kali:~$ ls
backup.txt renamed.txt

4 System Info & Management

Monitoring your system is crucial for performance and security. These commands help you gather system information:

Command Description
whoami Show current user
id Show UID/GID
uname -a Kernel and system info
top View running processes
htop Enhanced top (press q to quit)
df -h Show disk usage
free -h Show memory usage
uptime System running time

5 Networking Tools

Networking commands are particularly important for cybersecurity and penetration testing:

Command Description
ifconfig Show network interfaces (use ip a on newer systems)
ip a Show IP address & interface info
ping google.com Check network connectivity
traceroute google.com Trace path to a host
netstat -tuln Show open ports
nmap <IP> Network scan (requires nmap)
curl <url> Get web content from terminal
wget <url> Download file from web

6 User & Permission Commands

Managing users and permissions is crucial for system security:

Command Description
sudo <cmd> Run command as superuser
adduser <name> Add a new user
passwd Change password
chmod +x script.sh Make file executable
chown user:group file Change file owner
Permissions Example
kali@kali:~$ echo '#!/bin/bash' > script.sh kali@kali:~$ echo 'echo "Hello, World!"' >> script.sh kali@kali:~$ ls -l script.sh
-rw-r--r-- 1 kali kali 31 Aug 15 14:23 script.sh
kali@kali:~$ chmod +x script.sh kali@kali:~$ ls -l script.sh
-rwxr-xr-x 1 kali kali 31 Aug 15 14:23 script.sh
kali@kali:~$ ./script.sh
Hello, World!

7 Hacking Tools Usage

Kali Linux comes pre-installed with many security tools. Here are some basic commands to get started:

Command Description
nmap <target> Scan target IP/network
airmon-ng Manage Wi-Fi monitor mode
airdump-ng Capture Wi-Fi traffic
hydra Brute-force login tool
msfconsole Start Metasploit Framework
sqlmap -u <url> --dbs Auto SQL injection test
Important: Only use these tools on systems you own or have explicit permission to test. Unauthorized scanning or testing is illegal.

8 Package Management

Kali Linux uses APT (Advanced Package Tool) for package management:

Command Description
sudo apt update Update package list
sudo apt upgrade Upgrade packages
sudo apt install <pkg> Install new package
sudo apt remove <pkg> Remove package
Package Management Example
kali@kali:~$ sudo apt update
Hit:1 http://kali.download/kali kali-rolling InRelease
Reading package lists... Done
Building dependency tree... Done
All packages are up to date.
kali@kali:~$ sudo apt install -y nmap
Reading package lists... Done
Building dependency tree... Done
nmap is already the newest version (7.92+dfsg1-2).
0 upgraded, 0 newly installed, 0 to remove and 0 not upgraded.

9 Example Pentest Setup

Here's a basic sequence of commands you might use when starting a penetration test:

Basic Pentest Workflow
kali@kali:~$ sudo apt update
[Package lists updated]
kali@kali:~$ ifconfig
[Network interfaces displayed]
kali@kali:~$ ping -c 3 192.168.1.1
[Ping results to confirm connectivity]
kali@kali:~$ nmap -sV 192.168.1.1
[Service version detection scan results]
kali@kali:~$ msfconsole
[Metasploit Framework console starts]

Practice Exercises

Try these exercises to reinforce your Linux command skills:

Create a directory called "security_test" and navigate into it
Create three empty files: recon.txt, scan.txt, and report.txt
List all files in the directory with detailed information (permissions, size, etc.)
Add some text to recon.txt using echo and redirection
Make a copy of recon.txt called recon_backup.txt
Check your system's memory and disk usage
Create a simple shell script that prints "Security scan complete"
Make the script executable and run it
Key Takeaways:
  • Linux command line proficiency is essential for cybersecurity professionals
  • Start with basic navigation and file operations before moving to more complex tools
  • Practice regularly to build muscle memory for common commands
  • Understanding permissions and user management is crucial for security
  • Kali Linux provides specialized tools that build on these fundamental commands
