What is Footprinting?

Footprinting is the process of gathering information about a target system or network to identify potential vulnerabilities. It's like gathering blueprints before attempting to break into a building.

Passive Footprinting

Gathering information without directly interacting with the target system.

  • WHOIS lookups
  • DNS enumeration
  • Social media profiling
  • Google dorking

Active Footprinting

Directly interacting with the target to gather information.

  • Port scanning
  • Ping sweeps
  • Traceroute
  • Banner grabbing
Legal Notice: Always ensure you have proper authorization before performing any footprinting activities. Unauthorized reconnaissance can be illegal and may result in serious legal consequences.

Google Dorking Examples

Google dorking uses advanced search operators to find specific information that might not be easily accessible.

File Type Searches

site:example.com filetype:pdf

Find PDF files on a specific domain

Directory Listings

intitle:"Index of" site:example.com

Find open directory listings

Login Pages

inurl:admin site:example.com

Find admin login pages

Configuration Files

filetype:config site:example.com

Find configuration files

OSINT (Open Source Intelligence) Techniques

OSINT involves gathering information from publicly available sources. This is completely legal and forms the foundation of most reconnaissance activities.

Social Media

LinkedIn, Twitter, Facebook profiles

News & Articles

Press releases, news articles

Company Info

Annual reports, job postings

Search Engines

Google, Bing, specialized engines

Public Databases

Shodan, Censys, public records

Code Repositories

GitHub, GitLab, code leaks

Best Practice: Always combine multiple OSINT sources for comprehensive intelligence gathering. Cross-reference information from different sources to verify accuracy.
