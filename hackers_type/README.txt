Hacker Type Images

This folder contains images for different types of hackers to be used in the Ethical Hacking Basics page.

Please add the following images to this folder:

1. white_hat.jpg - An image representing a White Hat hacker (ethical hacker)
2. black_hat.jpg - An image representing a Black Hat hacker (malicious hacker)
3. grey_hat.jpg - An image representing a Grey Hat hacker (middle ground)

Recommended image specifications:
- Square or 4:3 aspect ratio
- Minimum size: 300x300 pixels
- File formats: JPG, PNG, or WebP
- Cartoon or illustrated style preferred for educational context
